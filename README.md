##  Expense Tracker (Python + SQLite)
A simple and efficient CLI-based Expense Tracker built using Python, SQLite, and Pandas.  
This project helps users track their daily expenses, categorize spending, and view summaries with clean data outputs.
##  Features
- Add new expenses with:
  - Amount  
  - Category  
  - Date  
  - Description  
- View all expenses  
- Filter by:
  - Date  
  - Category  
- Update or delete existing records  
- Persistent storage using SQLite database
- Clean tabular output using Pandas  
- Lightweight and beginner-friendly project
## Technologies Used
- Python 3
- SQLite
- Pandas
- CLI Interface
